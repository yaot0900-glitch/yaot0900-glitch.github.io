# 第十期：用户指定色板全面替换

## 背景

用户提供4个精确RGB色，要求全面替换现有配色。纯色不透明，不用毛玻璃。

---

## 一、色板定义

| # | RGB | Hex | 角色 |
|---|-----|-----|------|
| 🔵 亮天蓝 | `rgb(92,213,254)` | `#5CD5FE` | 全局背景、倒计时卡片 |
| 🔷 深蓝 | `rgb(34,55,107)` | `#22376B` | 正文文字、暗底卡片 |
| 🟫 深棕 | `rgb(105,60,30)` | `#693C1E` | 通用边框、暗色强调 |
| 🟠 暖橙金 | `rgb(240,178,99)` | `#F0B263` | 信誉值卡片、按钮、强调色 |

---

## 二、各元素配色分配

| 元素 | 配色 |
|------|------|

| 正文文字 | `#22376B` 深蓝 |
| 次要文字 | `#22376B` 70%透明度 |
| 通用边框 | `#693C1E` 深棕 |
| 按钮 primary | `#F0B263` 暖橙金（渐变到更亮） |
| 按钮 secondary | 白底 + `#693C1E` 边框 |
| `.glass-card` | 纯白 `#FFFFFF` + `#693C1E` 边框 |
| `.card-countdown` | `#5CD5FE` 20%淡蓝底 + `#693C1E` 边框 |
| `.card-credibility` | `#F0B263` 20%暖金底 + `#693C1E` 边框 |
| `.card-info` | 白色 + `#693C1E` 边框 |
| `.card-dark` | `#22376B` 深蓝底，白/金色文字 |
| `.card-question` | 纯白 + `#693C1E` 边框 |
| hover 发光 | `#F0B263` 暖橙金 glow |
| 判断按钮-真 | 绿底 |

---

## 三、改动文件

| 文件 | 改动 |
|------|------|
| `src/index.css` | 全面替换色板tokens、卡片类、按钮、面板 |
| `src/components/Button.tsx` | judge-false 改用暖橙金 |
| 其余文件 | 无需改动（CSS变量自动生效） |

---

## 四、执行顺序

1. `src/index.css` — 全量替换色板+卡片+按钮
2. `src/components/Button.tsx` — judge-false 配色
3. TypeScript 验证
